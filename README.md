Welcome to HearthOS, a small cozy Operating System built for ComputerCraft.  
To install HearthOS on your own CC:T Advanced Computer, all you need to do is run this simple command:  

wget https://raw.githubusercontent.com/Weaponized-Toaster/HearthOS/refs/heads/main/startup.lua startup.ua  

Once you have done that, you can then run  

reboot  
  
And HearthOS will begin setting itself up!
